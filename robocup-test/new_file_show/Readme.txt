fileshow
用于显示图片并调节阈值
常用方法如下

若只跟文件名，则为显示图片。

参数： -r
用法举例：./file_show -r colors0.txt 6_1.yuv
功能：通过手动选取属于颜色分类的点，从而改变当前颜色的阈值

左键点击：点击属于当前颜色分类的点，调整所有的阈值（y, u, v）
右键点击：点击不属于当前颜色分类的点，只调整选定的阈值（y, u, v中的一个）
键盘 i：改变当前颜色分类（分类号-1）
键盘 k：改变当前颜色分类（分类号+1）
键盘 q：撤销（只能撤销一步，撤销到最后一次点击前）
键盘 s：保存，如上例中，保存到colors0.txt中

键盘 y, u, v：未变，选定需要调整的阈值
键盘 1, 2, 3, 4：未变，改变阈值
